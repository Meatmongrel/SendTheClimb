Hi there, this project was spawned from my love of climbing, which I found only earlier this year. This app allows you to search for climbs within a certain proximity with a set difficulty range. The points on the map also give more information about the given climb and links to a page with further information. I used the MountainProject API run by REI in tandem with Google Maps API.
